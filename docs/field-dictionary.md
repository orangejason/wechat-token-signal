# 字段缩写对照字典

本文档列出所有中文缩写字段的完整含义，方便开发者快速查阅。

## 微信社群相关

| 缩写 | 全称 | 英文 | 说明 |
|------|------|------|------|
| `sfr` | 首发人 | First Launcher | 最早在微信群推荐该代币的人 |
| `sfsj` | 首发时间 | First Launch Time | 代币首次在微信群出现的时间 |
| `sfqy` | 首发社群 | First Launch Community | 代币最早出现的微信社群 |
| `sfzf` | 首发至今倍数 | First Launch Multiple | 从首发价到当前价的倍数 |
| `fshzf` | 发射后最高倍数 | Post-Launch Peak Multiple | 发射后达到的历史最高倍数 |
| `sfxx` | 首发信息 | First Launch Info | 首发人、倍数、市值变化的汇总 |
| `zfzgzf` | 至今最高倍数 | Peak Multiple to Date | 最高倍数含文字描述 |

## 社群热度

| 缩写 | 全称 | 英文 | 说明 |
|------|------|------|------|
| `cazs` | 查询指数 | Query Index | 被查询的热度指标 |
| `sqzs` | 社群指数 | Community Index | 在微信社群中的热度 |
| `bqfc` | 本群分享次数 | Group Share Count | 在当前群的分享次数 |
| `qwfc` | 全网分享次数 | Network Share Count | 全部微信群的分享总次数 |
| `fgq` | 覆盖群数 | Coverage Groups | 该代币被分享到的群总数 |

## 查询人相关

| 缩写 | 全称 | 英文 | 说明 |
|------|------|------|------|
| `cxr` | 查询人 | Querier | 最近查询该代币的用户 |
| `cxrxx` | 查询人信息 | Querier Info | 查询人的详细信息 |
| `cxrzf` | 查询人至今涨幅 | Querier Return | 查询后到当前的涨幅 |
| `zgzf` | 最高涨幅 | Peak Return | 查询后的最高涨幅 |
| `grcxcs` | 个人查询次数 | Personal Query Count | 该用户查询该代币的次数 |

## 持仓分布

| 缩写 | 全称 | 英文 | 说明 |
|------|------|------|------|
| `zbbl` | 占比比例 | Holding Ratio | 前 10 大持仓地址各自占比（`\|` 分隔） |
| `zzb` | 总占比 | Total Ratio | 前 10 大持仓合计占比（%） |

## 时间与状态

| 缩写 | 全称 | 英文 | 说明 |
|------|------|------|------|
| `dqsj` | 当前时间 | Current Time | 数据快照的时间戳（UTC+8） |
| `jy` | 交易 | Trade | 交易标记 |
| `xwb` | 新闻播报 | News Broadcast | 相关新闻播报内容 |

## 微信群信息

| 缩写 | 全称 | 英文 | 说明 |
|------|------|------|------|
| `qun_name` | 群名 | Group Name | 首发微信群名称 |
| `qun_id` | 群ID | Group ID | 微信群唯一标识 |
| `wx_id` | 微信ID | WeChat ID | 微信数据 ID |
| `qy_wxid` | 群友微信ID | Member WeChat ID | 首发群友的微信 ID |
| `qy_name` | 群友名称 | Member Name | 首发群友的昵称 |
